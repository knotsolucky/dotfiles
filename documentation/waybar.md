# Waybar

`config/waybar/config.jsonc` is **display-only**: no `on-click` / scroll handlers. Modules still show workspaces, window title, MPRIS, CPU, memory, audio, network, clocks, and the power label.

There is **no** `config/waybar/scripts/` directory in this repo; launch Waybar with `waybar` (or your own wrapper). Tray-only mode uses `tray-only.jsonc` + `tray-only.css`.
