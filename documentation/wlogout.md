# wlogout

## Hyprland window size

GTK buttons use **expand** inside a fullscreen layer, so they grow with the monitor. **`config/hypr/hyprland.conf`** forces **wlogout** to **float**, **center**, and a fixed **`size`** so the grid stays compact. Change **`size`** (e.g. `560 320`) if you add buttons or want a different footprint.

## Theme + launch

See repo **`config/wlogout/`** and Waybar **`custom/power`** `on-click` in **`config/waybar/config.jsonc`**.
